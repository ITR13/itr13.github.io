Find Luigi!

A recreation of the game "Wanted" from Super Mario 64 DS, with additional patterns and challenges.

Instructions

The top part of the screen will show a poster with the face of the person you need to find. Locate this person on the bottom screen and click them.
You start with 10 seconds to locate them, and gain 5 seconds each time you find the wanted person. If you click the wrong person you lose 10 seconds, so choose wisely!
If you fail to locate the wanted person, you can choose to continue playing. Each time you continue you'll be set back more and more levels though!

The further you get, the more difficult the levels become. The easy levels end at level 100.
Some levels will try to trick you, or force you to use reason, so beware!

If you need to pause the game you can switch to another tab.
The game also saves your progress so you can continue from where you stopped playing next time if you wish to :)

Want to help?

Most sprites and sounds are ripped from the game except the following. If you can make a better replacement for them, please send a PR:

  • Clock Ticking Sound
  • Gaining a star sound
  • Stars merging sound
  • "Correct" Sound Effect
  • Drumroll Short / end of Drumroll Long
  • Time increasing sound

I would also not mind more levels if you have any interesting ideas.
